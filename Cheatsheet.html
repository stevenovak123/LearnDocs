Basic Syntax​

These are the elements outlined in John Gruber’s original design document. All Markdown applications support these elements.

Headings​

# H1
## H2
### H3

Font Weights​

**bold text**

*italicized text*

> blockquote

Lists​

Ordered List

1. First item
2. Second item
3. Third item

Unordered List

- First item
- Second item
- Third item

Code snippets​

`code` or ```language``` or ```language title=filepath```

Horizontal Rule​

---
